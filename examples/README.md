# The Gallery

Each file in this folder is a complete worked example: real code in some
language, followed by its Pseudo translation, followed by a short note on
what the translation makes visible that the original hides.

The examples are deliberately drawn from different languages and different
kinds of logic, because that is the whole point - Pseudo reads the same no
matter what it was translated from.

| Example | Source language | What it shows |
|---|---|---|
| [Debounce](01-javascript-debounce.md) | JavaScript | Cutting through closure and callback syntax |
| [LRU cache](02-python-lru-cache.md) | Python | Making invisible state changes visible |
| [Top customers query](03-sql-window-query.md) | SQL | Pseudo works on declarative languages too |
| [Agent loop](04-agent-loop.md) | (design, no source) | Pseudo as a design language before code exists |

To reproduce any of these yourself, install the skill from the repo README
and ask your agent: "Translate this to Pseudo."
